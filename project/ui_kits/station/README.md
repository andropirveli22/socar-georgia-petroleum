# Station UI Kit — SOCAR Georgia Petroleum

Digital signage and pump-screen recreations for SOCAR fuel stations.

## Components
- `PriceBoard.jsx` — large-format LED price display (canopy/totem)
- `PumpScreen.jsx` — fueling pump terminal during a transaction
- `Wayfinding.jsx` — entry-canopy product/service indicators

## Run

Open `index.html` to see all surfaces side by side.
