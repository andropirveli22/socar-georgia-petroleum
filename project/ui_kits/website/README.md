# Website UI Kit — SOCAR Georgia Petroleum

A high-fidelity recreation of the SGP corporate website, modeled after sgp.ge / socar.ge.

## Components
- `Header.jsx` — sticky top nav with logo, primary nav, language switcher, CTA
- `Hero.jsx` — full-bleed hero with photo overlay
- `StatsBar.jsx` — 4-up tabular-numeral stats
- `ServicesGrid.jsx` — services / fuel-types card grid
- `StationFinder.jsx` — search + map placeholder
- `NewsRow.jsx` — news/press releases
- `Footer.jsx` — dark footer, 4 columns

## Run

Open `index.html`. It demonstrates a click-through landing experience with all components.
